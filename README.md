# Blog Website

Personal blog website of Jessica Karelle, computer science student.

## Features

- Responsive design
- Articles with category filtering
- Smooth scrolling navigation
- Clean and modern interface
- Sticky navigation bar

## Technologies

- HTML5, CSS3, JavaScript
- Bootstrap 5
- Font Awesome, Google Fonts

## Tools

- Git
- Visual Studio Code
- Phcode.dev

## Getting Started
```bash
git clone https://github.com/jessicakarelle/blog-website
cd blog-website
start index.html
```

Open `index.html` in your browser or visit "blog.jessicakarelle.com".
