# Portfolio Website

Personal blog website of Jessica Karelle, computer science student.

## Features

-   Responsive design
-   Articles
-   Smooth scrolling navigation
-   Contact section

## Technologies

-   HTML5, CSS3, JavaScript
-   Bootstrap 5
-   Font Awesome, Google Fonts

## Tools

-   Git
-   Visual Studio Code
-   Phcode.dev

## Getting Started

```bash
git clone https://github.com/jessicakarelle/jessicakarelle-portfolio
cd portfolio-website
start index.html
```

Open `index.html` in your browser or just search for "blog.jessicakarelle.com".
